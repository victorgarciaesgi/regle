# Regle + Vite

Exported from [Regle Playground](https://play.reglejs.dev). Requires [Node.js](https://nodejs.org) 18+ or 20+.

```sh
npm install
npm run dev
```

With pnpm:

```sh
pnpm install
pnpm run dev
```

Add optional packages used in the playground (e.g. `@regle/schemas`, `zod`, `valibot`) with your package manager if your `App.vue` imports them.
