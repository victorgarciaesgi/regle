# REGLE_R0003: Incorrect rule format

Code: `REGLE_R0003`
Level: error

## What this error means

Regle reported **Incorrect rule format** while running validation. This diagnostic helps you find misconfigured rules, state, or integration setup before it silently breaks form behaviour.

## Why this happens

- The runtime detected an unexpected value or configuration related to this area (Runtime).
- See the terminal message for the exact path, rule name, or parameter values involved.
