# REGLE_C0101: Nuxt setup file not found

Code: `REGLE_C0101`
Level: error

## What this error means

Regle reported **Nuxt setup file not found** while running validation. This diagnostic helps you find misconfigured rules, state, or integration setup before it silently breaks form behaviour.

## Why this happens

- The runtime detected an unexpected value or configuration related to this area (Nuxt).
- See the terminal message for the exact path, rule name, or parameter values involved.

## How to fix it

Follow the `fix:` hint in the terminal output. The diagnostic catalog in `@regle/nuxt` defines the recommended action for this code.

``ts
// nuxt.config.ts
export default defineNuxtConfig({
  modules: ['@regle/nuxt'],
  regle: { setupFile: './app/regle.config.ts' },
})
``
