# REGLE_R0101: Non-numeric rule parameter

Code: `REGLE_R0101`
Level: warn

## What this error means

Regle reported **Non-numeric rule parameter** while running validation. This diagnostic helps you find misconfigured rules, state, or integration setup before it silently breaks form behaviour.

## Why this happens

- The runtime detected an unexpected value or configuration related to this area (Rules).
- See the terminal message for the exact path, rule name, or parameter values involved.

## How to fix it

Follow the `fix:` hint in the terminal output. The diagnostic catalog in `@regle/rules` defines the recommended action for this code.

``ts
// Wrong
minValue('5')

// Correct
minValue(5)
``
